<ul>aide teamwork w/ github :
<li> git add (nom du dossier)</li>
<li> git clone (adresse) (rename)</li>
<li> git commit -m "(commentaire)"</li>
<li> git push /*envoyer l'information*/</li>
<li> git status /*fichier en attente*/</li></ul>
